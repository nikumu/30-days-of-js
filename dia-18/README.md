Dado um array de números, use o método `filter` para obter somente os números pares.

>O método filter() cria um novo array com todos os elementos que passaram no teste implementado pela função fornecida.

**Teste 1:**

Entrada: `2, 5, 7, 9, 12`

Saída: `2, 12`

**Teste 2:**

Entrada: `5, 6, 7, 8, 22`

Saída: `6, 8, 22`
